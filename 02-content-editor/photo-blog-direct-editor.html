1

Proje klasörünü seçin

GitHub Desktop'ta gördüğünüz proje klasörünü (içinde 01-website-ready-to-upload olan) seçin. Editör değişiklikleri bu klasöre yazar; böylece GitHub Desktop'ta hemen görünür.

2

İçeriği düzenleyin

Fotoğraf, hikâye, filtre, okuma süresi, slug ve metadata alanlarını düzenleyin. Kaydet'e bastığınızda hepsi otomatik senkronlanır ve görsel varyantları üretilir.

3

GitHub ile yayınlayın

Kaydettikten sonra GitHub Desktop'ta Commit + Push deyin. GitHub Actions siteyi otomatik derleyip yayına alır (~1-2 dk). Ek bir paket üretmenize gerek yok.
